# PC Builder AI Backend

Backend API for an AI-powered PC Builder platform.

# Features

* Product Management
* Category Management
* PC Compatibility Checker
* AI Build Recommendation
* PostgreSQL Database Integration

# Tech Stack

* NestJS
* TypeScript
* Prisma ORM
* PostgreSQL

# Key Functionalities

# Product Catalog

Manage PC components including CPU, Motherboard, GPU, RAM, SSD, and PSU.

# Compatibility Checker

Validate compatibility between selected components.

# AI Recommendation

Generate recommended PC builds based on user requirements and budget.

# REST API

Provide structured APIs for frontend integration.
